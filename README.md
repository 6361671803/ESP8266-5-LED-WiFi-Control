# ESP8266 5 LED Wi-Fi Control

This project demonstrates wireless control of 5 LEDs using ESP8266 connected to a Wi-Fi network.

## Components Used
- ESP8266 (NodeMCU / ESP-12E)
- 5 LEDs
- 5 × 220Ω resistors
- Breadboard and jumper wires

## Working
- ESP8266 connects to Wi-Fi using SSID and password
- IP address is displayed in Serial Monitor
- User opens IP address in browser
- LEDs are controlled via web page

## Steps to Use
1. Upload code to ESP8266
2. Open Serial Monitor at 115200 baud
3. Note the IP address
4. Open IP in browser
5. Control LEDs
